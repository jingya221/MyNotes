## 🚀 如何使用

1. 在 `notes/` 文件夹中创建新的markdown文件
2. 运行更新脚本来自动更新此README的目录索引
3. 提交并推送到GitHub

## 📋 自动更新目录

使用以下命令自动更新笔记目录：

```bash
python update_readme.py
```

或者使用批处理文件：

```bash
./update_notes.bat
```
