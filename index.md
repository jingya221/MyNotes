## 🚀 如何使用

### 浏览笔记
- 使用左侧导航栏按分类浏览笔记
- 使用顶部搜索框快速查找内容
- 点击笔记标题进入详细页面

### 添加新笔记
1. 在 `notes/` 文件夹中创建新的markdown文件
2. 在文件开头添加Front Matter（页面元数据）
3. 运行更新脚本自动生成导航

### 更新索引
```bash
# 使用Python脚本
python update_readme.py

# 或使用批处理文件
./update_notes.bat
```
